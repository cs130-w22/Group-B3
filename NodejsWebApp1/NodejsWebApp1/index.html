What is the Digital Humanities Project?

Our project is to develop a web application that assists the humanities department parse historic documents for images and text. For researchers, the image extracting and document processing is time consuming and largely done manually. With this application, this step is automated saving valuable resources and time. Our goal for this project is for the application to receive images of historic documents such as book pages or newspapers to the web application, recognize and then extract the text and images using machine learning, and then send an archive of cropped images and interpreted text to the users.
