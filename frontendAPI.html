Our frontend is composed of the following files and directories, located in
the 'NodejsWebApp1/NodejsWebApp1' subdirectory

  • server.js
  • index.html
  • install.sh
  • public/css/styles.css

More detailed descriptions of each is found below:
  • server.js
    This file contains the code to run the application server and contains all its functionality.

  • index.html
    This file is the HTML file for the application.

  • install.sh
    This file is the bash script that installs all the necessary libary dependencies and then calls server.js to run the application

  • styles.css
    This file works in conjuction to index.html and server.js that adds style attributes to the relevant element.
